# BloxFruits God

This repository contains scripts and executables for the BloxFruits game on Roblox. Please ensure you follow the usage guidelines carefully.

## Files
- **godclient.lua**: The main client script.
- **UltimateDeploy.bat**: A batch file for deployment.
- **RobloxGodExecutor.exe**: The executor for running scripts.
- **BuildRobloxExecutor.bat**: A batch file to build the Roblox executor.
- **assets/**: Contains image files used in the project.
